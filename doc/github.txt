github.com에 가서 login 후 new repository 를 클릭하여 새 저장소를 만든다.
새 저장소의 이름은 dbdb이다
- repository name: dbdb, public check, readme check, ...
- create repository 클릭
- 새 저장소의 주소는  https://github.com/shinsc3/dbdb 이다

< 내 컴퓨터의 내용을 repository로 보내기>
- git 실행
- project path로 이동(c:\terrysql\dbdb)
- git init (.git 디렉토리가 생긴다) : 버전 관리하겠다고 알려줌

- git status :  버전관리 안되는 것은 붉은색
- git add public
- git add src
- git add package.json 식으로 버전 관리하고 싶음 파일/디렉토리만 선택하여add

- git config --global user.name terry
- git config --gloabl user.emai shinsc3@hotmail.com

- git commit 하여 i 키를 입력하여 v0.1 어쩌구 저쩌구 작성 :wq 저장 종료
- git commit -m "initial commit"

- git remote add origin https://github.com/shinsc3/dbdb.git
 = remote 저장소를 add 하는데 그주소는 https://github.com/shinsc3/dbdb.git 이다
- git push -u origin master
 = 저장소에 push 한다
- git push(나중에  push할 경우)

- 원격 저장소를 지우고싶으면 git remote remove origin
